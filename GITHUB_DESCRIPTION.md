# GitHub Repository Description & Topics

## Description（描述）

### 中文版本
與 AI 導遊 Ailisha 一起環遊世界的互動式旅行遊戲。使用 Google Gemini API 自動生成城市照片、景點合照和旅行日記，創造專屬的虛擬旅行回憶。

### English Version
An interactive travel game exploring the world with AI guide Ailisha. Uses Google Gemini API to automatically generate city photos, landmark selfies, and travel diaries, creating unique virtual travel memories.

## GitHub 設定說明

### Description 設定位置
1. 前往 GitHub 專案頁面
2. 點擊右側的 ⚙️ Settings
3. 在 "About" 區塊中編輯 "Description"

### Topics 設定位置
1. 前往 GitHub 專案頁面
2. 點擊右側的 ⚙️ Settings
3. 在 "About" 區塊中編輯 "Topics"
4. 輸入標籤並按 Enter 確認
